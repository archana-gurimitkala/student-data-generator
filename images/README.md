# Screenshots

Add the following screenshots here:

1. **openai_output.png** - Screenshot showing the interface with OpenAI checkbox checked and generated data
2. **huggingface_output.png** - Screenshot showing the interface with OpenAI checkbox unchecked (using Hugging Face) and generated data
3. **interface_overview.png** - Screenshot showing the full interface with examples

## How to take screenshots:

1. In your Gradio interface, take a screenshot with the checkbox **checked** (OpenAI) → save as `openai_output.png`
2. Take a screenshot with the checkbox **unchecked** (Hugging Face) → save as `huggingface_output.png`
3. Take a general overview screenshot → save as `interface_overview.png`

Save all screenshots in this `images/` folder.
